Question 1
----------
        
What numbers does the following program print:  
`x = 3  
def f():  
&nbsp;&nbsp;&nbsp;&nbsp;x = 4  
&nbsp;&nbsp;&nbsp;&nbsp;print x  
f()  
print x`  

### Answer

4, 3

### Explanation

The x inside the function f is a local variable and does not affect the global x.
